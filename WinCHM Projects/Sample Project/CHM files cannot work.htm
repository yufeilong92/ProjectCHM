For security reason, Microsoft has updated Windows® system. With the new introduction of Microsoft Security Update, you CANNOT:

  • Run CHM files on network driver
  • Run CHM files that downloaded with Internet Explorer directly

These affect all HTML Help files, not only those created with WinCHM!

Refer to: http://www.microsoft.com/technet/security/bulletin/ms05-026.mspx
