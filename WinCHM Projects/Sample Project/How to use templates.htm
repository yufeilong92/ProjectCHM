Template is a html file that contains some special tags. Template can contain images, js, css files, etc. Template files must be stored in Project Folder or its subfolder.

Template is very useful to help you create uniform style help files. But it's not necessary when creating a help file.

Add prebuilt template.

WinCHM has several prebuilt templates that can be used immediately after adding.

  1. Click  or right-click Templates in the File list pane, select Add templates... on popup menu.
  2. The Add new topics dialog will appear.

     
  3. Check Prebuilt template radio box. Select a prebuilt template you like in the drop-down list box on the left of the radio box.
  4. Click on OK.
  5. You have to specify a folder to store the template file. If there is a file that has the same file name in the folder, you have to specify another folder name.
  6. Edit the template just like editing a html page. You can edit the letters, change images, change table background colors, etc.
  7. Click on to save the template.

Apply a template.

To use a template, you must apply it at first.

Right-click a template name in File list pane, select Set as HTML help template or Set as web help template on the popup menu. You can also set these in Project setting.
