To integrate Web Help into your existing web site:

1. Build Web Help. (Build > Web help...)

2. Upload all of the files in the Web Help output folder (You can click View > Web help folder to open this folder) into a folder on your web hosting. Generally we use FTP software to upload files.

3. In your existing web page, create a hyperlink that links to index.htm .

For example:

Assume your web site address is http://www.myweb.com and you have uploaded the Web help into folder /help. The web help address will be http://www.myweb.com/help/index.htm.

If necessary, please consult your web designer and web hosting service provider.
