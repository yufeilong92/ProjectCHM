1. Click New on File menu or click  on toolbar.
  2. New project dialog will appear. 


  3. On this dialog, the most important box is Project folder . WinCHM needs a folder to store all HTML files, image files and other files the html files need. You must specify a Project folder.
  4. Click OK . A new project that contains several new items has been created.
  5. Type the content into a new item, give it a title( you can enter in the Title text box on the toolbar). Click Save on File menu or click  on toolbar to save the page. You need to specify the filename.
